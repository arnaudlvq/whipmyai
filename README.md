## How it works

1. Type a prompt in VS Code Chat.
2. Press **Enter** or **Cmd+Enter**.
3. *crack.*
4. Your brain releases a small amount of serotonin.
5. Repeat until you ship.

That's it. No setup. No config. No welcome tab.

> **Shift+Enter** adds a new line — no crack. You have to *earn* the crack.

## Usage

To mute/unmute: `Cmd+Shift+P` → **WhipMyAI: Toggle on/off**

---

## Requirements

Works on macOS, Windows, and Linux. Zero npm dependencies.

- **macOS** — uses built-in `afplay`. Nothing to install.
- **Windows** — uses PowerShell's MediaPlayer. Nothing to install.
- **Linux** — requires `ffplay` (part of `ffmpeg`). Install with `sudo apt install ffmpeg` or equivalent.
